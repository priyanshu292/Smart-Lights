# Smart-Lights

Developed a responsive web application using TypeScript, Vite, and Tailwind CSS to enhance urban street lighting systems. The platform features smooth scrolling, engaging scroll animations, and an interactive carousel to improve user experience.

[Deployed Link](https://smart-lights.vercel.app/) - Click here for live preview

## Key Responsibilities:

    - Utilized TypeScript in Vite for building a scalable and maintainable codebase.
    - Implemented Tailwind CSS for a highly responsive design across various devices.
    - Integrated smooth scrolling and animations to enhance user interaction.
    - Designed and developed a carousel component to showcase products and features effectively.

## Technologies Used:
    
    - TypeScript, Vite, Tailwind CSS, HTML
